# DrSlon Blog Theme

Современная WordPress block theme для технического блога.

## Идея

Тема строится как лёгкая, читаемая и расширяемая база для блога про:

- WordPress
- Linux
- DevOps
- Nginx
- Docker
- Python
- безопасность
- серверную инфраструктуру

## MVP

- базовый theme.json
- header и footer
- index, home, single, page, archive, search, 404
- стартовые patterns
- базовые стили под технический контент
